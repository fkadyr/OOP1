1. Первая программа, это компьютерная игра The Elder Scrolls Skyrim.
    В нем есть персонаж класса воин. У воина есть атрибуты.
    Например:
        Класс Воин:
            Имя = kadir
            Уровень = 1
            Здоровье = 100
            Выносливость = 100
            Мана = 100
            Расса = Норд
            Навыки = [Одноручное оружие, Тяжеля броня, Кузнечное Дело, ...]
        У класса воин есть имя персонажа. А также есть уровень, при увиличении котрого, у персонажа
        можно увеличить здоровье, или выносливоть, или ману. Также при прокачивания уровня можно увеличивать
        Навыки нашего персонажа. Еще у нашего персонажа есть расса, при выборе котроко даются определенные способности.
        Например при выборе рассы Норд мы получаем защиту от холода, и тд.
2. Вторая программа, банковское приложение.
    В нем есть класс банковская карта. И у него есть атрибуты:
    Например:
        Класс Банковская Карта:
            Имя = Kadir
            Номер карты = 0000 0000 0000 0000
            Дата окончания срока службы = 01/2020
            CVV код = 111
            Пароль = 1111
            Номер телефона = 89393911111
            Баланс = 1000

        В классе банковская карта у пользователя есть имя. И у карты есть номер так нзываемый банковский идентификатор.
        А также у карты есть дата окончания срока службы. Также у карты есть CVV код, который служит для проверки подленности карты.
        Пароль при оплате через терминал. Номер телефона для быстрых переводов через телефон. Баланс - остаток счета нашей карты.
